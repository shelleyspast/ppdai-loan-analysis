# 拍拍贷平台借款数据分析 | PPDai Loan Data Analysis

## 项目简介
基于拍拍贷真实业务数据（2015-2017年），对平台借款业务进行探索性数据分析（EDA），
包括用户画像、借款趋势、还款行为及风险分析。

在原有分析基础上，延伸探索了**认证完整度与历史逾期率的关系**，
发现认证数量与逾期率呈正相关（与直觉相反），并提出可能的业务解释。

## 数据来源
拍拍贷公开业务数据样本，包含2015年1月至2017年1月间328,553支信用标。
数据仅供学习和分析使用，不代表平台整体情况。

## 主要分析内容
- 借款金额与借款期限的分布特征
- 用户画像分析（年龄、性别、借款类型）
- 借款金额月度趋势
- 还款行为与逾期风险分析
- 延伸分析：认证数量与历史逾期率的关系

## 核心发现
认证数量与历史逾期率呈正相关：认证数为0的用户逾期率最低（7%），
认证数为4的用户逾期率最高（41.5%）。
这一结果表明认证数量可能反映借款需求的迫切程度，而非还款能力，
平台风控设计应结合认证质量而非仅看数量。
